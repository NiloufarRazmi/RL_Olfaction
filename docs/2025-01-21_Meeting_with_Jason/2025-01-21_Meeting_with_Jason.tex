\begin{itemize}
\item Left/right task:
\begin{itemize}
\item With the Cartesian inputs silenced \(\to\) the agent can solve the task

\end{itemize}
\item East/west task:
\begin{itemize}
\item With the Cartesian inputs silenced \(\to\) the agent can solve the task
